# droid-libx

This a extension library for android,it contains several handy tools.

# How can I use it?

## Gradle

### Java

``` gradle
dependencies {
    implementation 'real.droid:libx-core:0.0.3'
}
```

### kotlin

``` gradle
dependencies {
    implementation 'real.droid:libx-core-ktx:0.0.3'
}
```
